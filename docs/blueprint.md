# **App Name**: ClubHive

## Core Features:

- Secure Authentication: Allow students to sign in via email or Google, using Supabase Auth.
- Club Dashboard: Enable students to view a 'My Clubs' page showing all clubs they belong to, along with associated events, announcements, and activities.
- Event Registration: Enable students to register for club events and generate a QR-based entry pass upon registration.
- Event Management: Allow club admins to create, manage, and promote events to club members. This will leverage Supabase database to save club related information and events.
- AI-Powered Event Reports: Generate AI-powered event reports using the Gemini API including attendance summaries, engagement insights, and department/year-wise participation using AI tool. The generated report should be saved for the admins to download. The information to generate these reports are stored in Supabase.
- Report Download: Allow club admins to download the generated event reports. It leverages the Supabase to upload and retrieve downloadable reports.
- Secure Access Control: Ensure students can only view their own data and club admins can manage only their respective clubs.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey professionalism and trustworthiness.
- Background color: Light blue (#D4E0F7), providing a clean and modern look with sufficient contrast against darker text elements.
- Accent color: Vibrant orange (#FF8C42) for calls to action, highlighting interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif font providing a clean, modern, objective and neutral feel suitable for both headlines and body text.
- Use a set of modern, minimalist icons for navigation and event representation. Icons should be consistent and easily recognizable.
- Design a clean and intuitive layout with clear sections for events, clubs, and user profiles. Ensure a responsive design for accessibility across devices.
- Use subtle animations for transitions and feedback, enhancing user experience without being distracting.
